# Projeto Lapidado - Guia de Execução ✨

Este projeto contém o sistema completo de catálogo inteligente para semijoias, composto por um aplicativo administrativo (Lojista) e um catálogo web (Cliente).

## 🚀 Como Rodar o Projeto

### 1. Requisitos Prévios
- Ter o **Node.js** instalado.
- Ter o **Expo Go** instalado no seu celular (para testar o App do Lojista).

---

### 2. Rodando o App do Lojista (Admin)
1. Abra o terminal na pasta `app-lojista`.
2. Execute o comando para instalar as dependências:
   ```bash
   npm install
   ```
3. Inicie o aplicativo:
   ```bash
   npm start
   ```
4. Escaneie o QR Code que aparecerá no terminal usando o aplicativo **Expo Go** no seu celular.

---

### 3. Rodando o Catálogo Web (Link da Bio)
1. Abra o terminal na pasta `catalogo-web`.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse o catálogo no seu navegador em: `http://localhost:3000`

---

## 💎 Funcionalidades Implementadas
- **IA Vision:** Reconhecimento automático de categoria e banho através da foto.
- **IA LLM:** Geração automática de descrições comerciais luxuosas.
- **Markup:** Cálculo automático de preço de venda baseado no custo.
- **White Label:** Customização de cores e templates de saudação.
- **WhatsApp:** Integração direta para fechamento de vendas com lista detalhada.

---

## 🛠️ Tecnologias Utilizadas
- **Mobile:** React Native & Expo.
- **Web:** Next.js & Tailwind CSS.
- **IA:** Integração baseada em Google Gemini (Vision & Pro).
- **Linguagem:** TypeScript para maior segurança e robustez.
